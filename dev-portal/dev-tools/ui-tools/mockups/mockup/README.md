# Mockup Google AI Studio (aislado)

Este mockup replica la experiencia visual y de interacción de Google AI Studio, usando React + Tailwind + Inter + FontAwesome, todo en una carpeta aislada para no afectar el código principal.

## ¿Qué incluye?
- Sidebar colapsable, minimalista, con iconografía elegante
- Header con menú usuario, campanita, accesos rápidos
- Panel derecho colapsable para settings/contexto
- Área central tipo chat/contenido
- Soporte dark/light mode (switch en el header)
- Fuente Inter y FontAwesome solo para el mockup

## ¿Cómo probarlo?

1. **Importa el mockup en tu App temporalmente:**

```tsx
import MockupAIStudioApp from './mockup-ai-studio';

function App() {
  return <MockupAIStudioApp />;
}
```

2. **Navega a la app y revisa el mockup.**

## Recomendaciones
- No mezcles este código con tu app principal. Úsalo solo como referencia visual o para migrar componentes a tu stack real.
- El mockup es 100% configurable y extensible.
- Puedes adaptar la fuente, iconos o colores fácilmente.
