# Mockup AI Studio

Este mockup es una interfaz de alta fidelidad inspirada en Google AI Studio, implementada en React + TypeScript + Tailwind CSS, con fuente Inter y Font Awesome para iconografía. 

## Características
- Layout de 3 columnas (sidebar colapsable, panel central, panel derecho colapsable)
- Header flotante con menú de usuario, notificaciones y cambio de tema
- Área central para chat/cards y barra de entrada fija
- Tema dual (dark/light) con animaciones sutiles
- Componentes modulares y reutilizables

## Visualización
1. Instala las dependencias necesarias:
   - Tailwind CSS
   - Font Awesome (`@fortawesome/react-fontawesome`, `@fortawesome/free-solid-svg-icons`)
   - Fuente Inter (importada en el CSS global o en el entrypoint)
2. Importa y renderiza `<MockupAIStudioApp />` en tu aplicación para ver el mockup.

## Notas
- El mockup es solo visual, no tiene lógica de negocio.
- Todo el código está aislado en `src/mockup-ai-studio/` para evitar interferencias.
