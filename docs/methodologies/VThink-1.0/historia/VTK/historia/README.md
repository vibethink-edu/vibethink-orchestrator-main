# Documentación Histórica y Obsoleta de la Metodología

Esta carpeta contiene documentos, manifiestos y artefactos de versiones anteriores de la metodología (XTR, XTP, etc.), conservados únicamente como referencia histórica y para trazabilidad organizacional.

## Política de manejo
- **No se utilizan estos documentos como base operativa ni para procesos activos.**
- Todo documento aquí está claramente marcado como **OBSOLETO** y referencia la metodología vigente (**VTK 1.0**).
- Se mantienen para:
  - Trazabilidad y memoria organizacional
  - Auditorías externas o internas
  - Aprendizaje y análisis de evolución metodológica
  - Justificación de migraciones y mejoras

## ¿Cuándo revisar lo histórico?
- Solo si una auditoría, revisión estratégica o investigación específica lo requiere.
- Si se detecta que alguna práctica valiosa no fue migrada a VTK 1.0 y se desea rescatar.

## Recomendación
Enfocar la mejora continua y la documentación viva en **VTK 1.0**. Lo histórico es solo para consulta puntual, nunca para operación diaria.
