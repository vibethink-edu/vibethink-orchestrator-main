# Glosario VThink Methodology 1.0

- **Evidencia:** Documento o artefacto que respalda el cumplimiento de una práctica o requisito.
- **Trazabilidad:** Capacidad de seguir el historial, aplicación y ubicación de todo artefacto o decisión.
- **Mejora continua:** Proceso sistemático de optimización y aprendizaje organizacional.
- **Memoria organizacional:** Conjunto de conocimientos, decisiones y aprendizajes acumulados por el equipo.
- **Multi-tenant:** Arquitectura que permite servir a múltiples clientes (empresas) de forma aislada y segura.
- **SaaS:** Software as a Service, modelo de entrega de software basado en la nube.
- **Auditoría:** Proceso de revisión formal para verificar cumplimiento de estándares y buenas prácticas.
- **Colaboración:** Trabajo conjunto y transparente entre todos los actores del proyecto.
