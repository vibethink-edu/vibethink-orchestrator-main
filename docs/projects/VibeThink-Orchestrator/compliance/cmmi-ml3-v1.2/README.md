# Evidencia y Documentación CMMI ML3

Este directorio contiene toda la evidencia, documentación y métricas requeridas para la certificación y auditoría bajo la metodología **CMMI Nivel de Madurez 3 (ML3)**, que es la metodología principal de este proyecto.

## Versión
- **CMMI ML3** (Maturity Level 3)

## Alcance
- Gestión de datos
- Empoderamiento de la fuerza laboral
- Procesos, métricas y prácticas requeridas por CMMI ML3
- Evidencia de cumplimiento para auditoría y mejora continua

## Estructura
- `01-data-management/` — Evidencia y documentación sobre gestión de datos
- `03-workforce-empowerment/` — Métricas y evidencia sobre empoderamiento de la fuerza laboral
- Otros subdirectorios según áreas CMMI relevantes

## Reglas
- No mezclar evidencia de otras metodologías o versiones en este directorio.
- Mantener la estructura y los nombres de archivos claros y auditables.
- Actualizar este README si se agregan nuevas áreas o cambia el alcance.

## Responsable
- Equipo de calidad y compliance
