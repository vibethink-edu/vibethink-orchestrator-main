# Blueprints de UI/UX y AI – e2crm

Esta carpeta contiene los blueprints, referencias visuales y funcionales para las interfaces clave del CRM e2crm, incluyendo el comportamiento esperado del Asistente de IA contextual.

## Propósito
- Documentar la estructura, layout y flujos principales de la interfaz del CRM.
- Especificar el comportamiento proactivo y contextual del Asistente de IA en cada vista.
- Servir como referencia para el equipo de producto, diseño, desarrollo y AI.

## Contenido
- `CRM_LAYOUT_BLUEPRINT.md`: Blueprint visual y funcional de las vistas principales del CRM y el asistente de IA.

## Buenas prácticas
- Mantén los blueprints actualizados conforme evolucione el diseño o las capacidades de IA.
- Agrega nuevos archivos para otros módulos, flujos o componentes relevantes.
- Documenta siempre el propósito y alcance de cada blueprint.
