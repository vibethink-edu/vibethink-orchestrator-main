# dev-dashboard

> **Nota:** Esta documentación cubre únicamente aspectos adicionales o específicos de `dev-dashboard`. La arquitectura general y los principios base son heredados de la arquitectura global de VibeThink Orchestrator. Consulta la documentación general para entender el contexto completo.

## Descripción

`dev-dashboard` es el portal de control y monitoreo para el equipo de desarrollo de VibeThink Orchestrator. Permite visualizar métricas, logs, herramientas de monitoreo y accesos rápidos a utilidades de desarrollo.

## Objetivos
- Centralizar información relevante para el equipo de desarrollo.
- Mejorar la eficiencia y trazabilidad de los procesos de desarrollo.
- Integrar herramientas de monitoreo, CI/CD y métricas de calidad.

## Estructura
- Panel de métricas
- Acceso a logs y auditoría
- Integración con herramientas de CI/CD
- Accesos rápidos a utilidades internas

## Ejemplo de uso
- Acceder al dashboard en entorno local: `http://localhost:3000`
- Consultar logs recientes desde el panel principal

## Responsables
- Equipo de desarrollo interno
